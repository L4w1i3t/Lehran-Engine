# nlohmann/json Header

This directory should contain the `json.hpp` header file from nlohmann/json.

## Download

Get it from: https://github.com/nlohmann/json/releases

Download the single-header version `json.hpp` and place it in this directory.

Or use this direct link:
https://github.com/nlohmann/json/releases/download/v3.11.3/json.hpp

## Quick Download (PowerShell)

```powershell
Invoke-WebRequest -Uri "https://github.com/nlohmann/json/releases/download/v3.11.3/json.hpp" -OutFile "json.hpp"
```
